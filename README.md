# EBQ Landing Page

A modern landing page for EBQ built with React 19, Vite, Tailwind CSS 4, and React Compiler.

## Features
- Responsive design
- Tailwind CSS for styling
- React 19 with React Compiler for optimized renders
- Vite for fast development and builds
- Event listeners: Magnetic Button and Progress bar on scroll

## Prerequisites
- Node.js (v20 or higher)
- npm (comes with Node.js)

Check Node version:
```bash
node --version
```
If not installed, download from [nodejs.org](https://nodejs.org/).

## Quick Start

1. **Clone or open the project**
   ```bash
   cd /path/to/React/EBQ-Landing-Page
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   This installs React, Tailwind CSS, and other dependencies.

3. **Start development server**
   ```bash
   npm run dev
   ```
   Opens at http://localhost:5173 (or similar).

4. **Edit and develop**
   - Main files: `src/App.jsx`, `src/components/`
   - Styles via Tailwind in `src/App.css` or components

## Build Commands

- **Development**: `npm run dev`
- **Lint code**: `npm run lint`
- **Build for production**: `npm run build` (outputs to `dist/`)
- **Preview build**: `npm run preview`

## Project Structure
```
.
├── public/          # Static assets (images, favicons)
├── src/
│   ├── components/  # Reusable components (Hero, About, CTA)
│   ├── App.jsx      # Main App
│   ├── App.css      # Global styles
│   └── main.jsx     # Entry point
├── package.json     # Dependencies & scripts
├── vite.config.js   # Vite + Tailwind + React Compiler config
└── tailwind.config.js (auto-generated by Tailwind)
```

## Troubleshooting
- **Port conflict**: Vite shows new port (e.g., 5174).
- **Tailwind issues**: Ensure `npm install` run; restart dev server.
- **React Compiler**: Enabled in vite.config.js; impacts perf but optimizes.
- **ESLint errors**: Run `npm run lint` or fix in VS Code.

## Deployment
